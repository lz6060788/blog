### C. API 接口测试

#### C1. 文章列表 API
- [ ] `GET /api/posts` 返回文章列表
- [ ] 支持分页参数：`?page=1&limit=10`
- [ ] 支持搜索：`?search=关键词`
- [ ] 支持筛选：`?published=true` 或 `?drafts=true`
- [ ] 响应格式包含分页元数据（total, page, totalPages, hasNext, hasPrev）

#### C2. 文章详情 API
- [ ] `GET /api/posts/[id]` 返回单篇文章
- [ ] 包含分类信息
- [ ] 包含标签列表（对象数组格式）
- [ ] 未认证返回 401

#### C3. 创建文章 API
- [ ] `POST /api/posts` 可创建文章
- [ ] 返回 201 状态码
- [ ] 自动生成摘要（如果未提供）
- [ ] 标签正确关联

#### C4. 更新文章 API
- [ ] `PUT /api/posts/[id]` 可更新文章
- [ ] 支持更新标签
- [ ] 更新成功返回最新数据

#### C5. 删除文章 API
- [ ] `DELETE /api/posts/[id]` 可删除文章
- [ ] 返回 204 状态码

#### C6. 统计 API
- [ ] `GET /api/admin/stats` 返回统计数据
- [ ] 所有统计字段都有值
