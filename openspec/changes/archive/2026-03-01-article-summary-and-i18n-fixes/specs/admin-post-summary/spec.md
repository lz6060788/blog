## ADDED Requirements

### Requirement: AI 摘要编辑器组件
系统应提供可复用的 AI 摘要编辑器组件，用于在文章新建和编辑页面显示和管理 AI 摘要功能。

#### 场景：组件在编辑页面正常工作
- **当** 用户在文章编辑页面
- **那么** 系统应显示 AI 摘要编辑器组件
- **并且** 组件应接收文章 ID 作为 prop
- **并且** 组件应加载现有的摘要内容（如果有）
- **并且** 组件应显示当前的摘要生成状态

#### 场景：组件在新建页面禁用状态
- **当** 用户在文章新建页面
- **那么** 系统应显示 AI 摘要编辑器组件
- **并且** 组件应接收 null 作为文章 ID
- **并且** 组件应显示提示信息，告知用户需要先保存文章
- **并且** 生成摘要按钮应处于禁用状态

### Requirement: AI 摘要生成
系统应允许用户为文章生成 AI 摘要。

#### 场景：生成摘要
- **当** 用户点击"生成摘要"按钮
- **并且** 文章标题和内容不为空
- **并且** AI 模型已配置
- **那么** 系统应调用摘要生成 API
- **并且** 系统应将摘要状态更新为"生成中"
- **并且** 系统应开始轮询摘要状态

#### 场景：生成前验证
- **当** 用户点击"生成摘要"按钮
- **并且** 文章标题或内容为空
- **那么** 系统应显示错误提示
- **并且** 系统不应调用摘要生成 API

#### 场景：AI 未配置提示
- **当** 用户点击"生成摘要"按钮
- **并且** AI 模型未配置
- **那么** 系统应显示提示信息，引导用户前往设置页面
- **并且** 系统不应调用摘要生成 API

### Requirement: 摘要状态轮询
系统应定期轮询摘要生成状态，直到生成完成或失败。

#### 场景：开始轮询
- **当** 摘要状态变为"生成中"
- **那么** 系统应每 3 秒轮询一次状态
- **并且** 系统应更新 UI 显示当前状态

#### 场景：生成成功
- **当** 轮询检测到摘要状态为"已完成"
- **那么** 系统应停止轮询
- **并且** 系统应显示生成的摘要内容
- **并且** 系统应显示成功提示
- **并且** 用户可以手动编辑摘要内容

#### 场景：生成失败
- **当** 轮询检测到摘要状态为"失败"
- **那么** 系统应停止轮询
- **并且** 系统应显示失败提示
- **并且** 系统应显示"重新生成"按钮

### Requirement: 摘要重新生成
系统应允许用户重新生成 AI 摘要。

#### 场景：重新生成确认
- **当** 用户点击"重新生成"按钮
- **那么** 系统应显示确认对话框
- **并且** 对话框应提示用户重新生成将覆盖现有摘要

#### 场景：执行重新生成
- **当** 用户在确认对话框中点击确认
- **那么** 系统应调用摘要生成 API
- **并且** 系统应重新开始状态轮询

### Requirement: 摘要手动编辑
系统应允许用户手动编辑 AI 生成的摘要。

#### 场景：编辑摘要内容
- **当** 摘要已生成
- **那么** 用户可以在文本框中编辑摘要内容
- **并且** 修改后的内容应在保存文章时一同保存

#### 场景：生成期间禁用编辑
- **当** 摘要状态为"生成中"
- **那么** 摘要文本框应处于禁用状态
- **并且** 系统应显示遮罩层提示用户等待
- **并且** 保存和发布按钮应处于禁用状态

### Requirement: 摘要状态显示
系统应在 UI 中清晰显示摘要的当前状态。

#### 场景：状态图标显示
- **当** 显示摘要状态
- **那么** 系统应根据状态显示对应的图标和颜色
- **并且** 未生成状态应显示时钟图标
- **并且** 生成中状态应显示旋转的加载图标
- **并且** 已完成状态应显示勾选图标
- **并且** 失败状态应显示错误图标

#### 场景：状态标签显示
- **当** 显示摘要状态
- **并且** 选择显示标签模式
- **那么** 系统应在图标旁显示文字标签
- **并且** 标签应说明当前状态（未生成/生成中/已完成/失败）

### Requirement: 新建页面工作流
系统应在新建文章页面提供自动生成摘要的选项。

#### 场景：显示自动生成选项
- **当** 用户在新建文章页面
- **那么** 系统应显示"保存后自动生成摘要"的勾选框
- **并且** 勾选框应有清晰的说明文字
- **并且** 勾选框默认未勾选

#### 场景：勾选自动生成后保存
- **当** 用户勾选"保存后自动生成摘要"
- **并且** 用户点击"保存草稿"或"发布"按钮
- **那么** 系统应保存文章
- **并且** 保存成功后跳转到文章编辑页面
- **并且** 系统应自动触发生成 AI 摘要
- **并且** 摘要状态应变为"生成中"

#### 场景：未勾选自动生成后保存
- **当** 用户未勾选"保存后自动生成摘要"
- **并且** 用户点击"保存草稿"按钮
- **那么** 系统应保存文章
- **并且** 保存成功后跳转到文章列表页面
- **并且** 不触发生成摘要
