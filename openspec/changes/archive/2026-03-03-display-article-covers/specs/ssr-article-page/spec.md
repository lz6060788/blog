## 修改需求

### 需求：文章详情页顶部显示封面
系统应在文章详情页顶部展示文章封面图片作为头图。

#### 场景：显示有封面的文章
- **当** 文章有封面图片时
- **那么** 系统应在文章内容顶部显示封面大图
- **并且** 封面比例应为 16:9
- **并且** 封面宽度应为页面全宽
- **并且** 封面最大高度不应超过视口的 60%

#### 场景：显示无封面的文章
- **当** 文章没有封面图片时
- **那么** 系统应跳过封面显示区域
- **或者** 系统可显示简洁的顶部装饰元素
- **并且** 不应影响文章内容的正常显示

#### 场景：封面图片优化加载
- **当** 文章详情页加载时
- **那么** 封面图片应使用优先加载策略
- **并且** 封面应在首屏尽快显示
- **并且** 系统应使用 Next.js Image 组件优化图片

#### 场景：封面 SEO 元数据
- **当** 文章有封面图片时
- **那么** 系统应生成 og:image 标签
- **并且** 系统应生成 twitter:image 标签
- **并且** 图片 URL 应为完整的绝对路径
